数当てゲーム

数当てゲームのページ:kazuate.html

数当てでは1〜10の乱数を生成し、3回目までに正解、4回目以降に正解すると答えとゲーム終了を告げるメッセージが表示されるようなプログラムを作りました 仕組みは単純で入力した値と乱数で生成された値を条件式で判断するということだったけれど、ボタンを配置、htmlに書かれている内容を上書きなどといった、新しく習ったことばかりを使っていたので、 作成をするのはかなり大変だった。でも、自分でWebサイトを作り、簡単なゲームを作ることができたことはとてもいい体験だと思いました。

各授業回の学習事項

第1回 WWW と HTML

第1回のWWWとHTMLではWebサイトを作るにあたって必要最低限なタグの使い方を学び実際に使用することでより深く理解をしていきました。 ハイパーリンクや画像の貼り付けなどといった少しでも達成感を得られる内容を学びました。これらのことを活かして、この授業での最終も目標であるWeb作成 を完成させたいです。

演習の成果:演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回の講義ではCSSというWebページの装飾を学びました。cssプログラムでhtmlで使ったタグを指定し、コマンドを打ち込むことでページの装飾ができるといったものでした

演習の成果:演習2-3: ページの見た目の装飾

第3回 JavaScript の基本

課題3-1では'yoso'の値を固定し、'kotae'の値を1~9の乱数で決めるということをやりました。条件分岐を描き、正解なら正解、不正解だった場合は答えが大小どちらかを表示するというプログラムを作りました。

演習の成果:演習3-6: JavaScriptの使い方

第4回 DOM と JavaScript

第4回の講義では、JavaScriptの続きと新しくDOMを学びました。DOMとはJavaScriptと合わせることで動きのあるWebページを作ることができ、要素の追加、消去や要素の変更などができるようになった。

演習の成果:演習4-2~5: DOMとJavaScript

第5回 イベント処理

第5回の講義ではイベント処理ということを学びました。イベント処理とはボタンを押したら画面の色が変わる仕組みや入力した文字を出力するといったWebサイトには必要なことを学びました

演習の成果:演習5-3: イベント処理
